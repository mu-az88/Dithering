**Dithering assignment**

In this assignment I coded three dithering techniques:
  1. Random Dithering
  2. Error Diffusion Dithering
  3. Uniform Dithering (Bayer Matrix) 



**Guidlines:**
1. Place your image files in the 'images' folder for the dithering script.
2. PNG or JPG formats are preferred
3. After adding an image, update line 79 in `Dithering code.py` to point to the chosen image (for example: `images/your_image.jpg`) and test the code on your own image.
